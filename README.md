## 📊 Funciones disponibles

### 1. **F_ESTM_RES** - Función de Cálculo de Fecha Estimada de Resolución
**Categoria:** Oracle PL/SQL  
**Propósito:** Calcular la fecha estimada en que una orden de trabajo debería estar resuelta, considerando solo días hábiles (excluye fines de semana y festivos nacionales).
